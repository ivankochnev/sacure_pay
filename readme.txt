READ ME. HOW INSTALL AND RUN CREATION OF DORMANT ACCOUNT PROJECT

I. :::::::::::::: Install Cypress on you PC ::::::::::::::

https://docs.cypress.io/guides/getting-started/installing-cypress#System-requirements


II. :::::::::::::: OPEN CYPRESS PROJECT :::::::::::::

 1. Download SecurePay project
 2. in console cd to SecurePay folder
 3. yarn install
 4. yarn cypress


III. :::::::::::::: RUN CYPRESS TESTS ::::::::::::::

 • command to run ALL test cases (feature files)  
 yarn cy:test 
 • command to open Cypress Console and run tests in it 
 yarn cy:open


Enjoy :)
